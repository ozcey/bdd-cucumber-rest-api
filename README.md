# BDD Framework for Testing Rest APIs

## Getting Started


### Prerequisites

- Java 8+
- Maven
- Cucumber 
- IntelliJ IDEA or Eclipse

### Setup


The project link here [GitHub link](https://github.com/ozcey/bdd-cucumber-rest-api) or  clone the project using the following command.
